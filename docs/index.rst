PoussePousseEditData
#########################

Introduction
==============

PoussePousseEditData est un plugIn QGis de création manuel de jeu de données géographiques sur fond cartographique. 

La création des objets ponctuels se fait à la manière du jeu pousse-pousse où l'on doit reconstituer une image en déplaçant des cases. 
Une grille recouvre le jeu de données ce qui permet de saisir des objets ponctuels cellule après cellule.
