Installation 
##############


PoussePousseEditData est un plugin pour le logiciel QGIS. Il est donc nécessaire que ce dernier soit installé sur l’ordinateur. 
QGIS peut être téléchargé gratuitement via l’URL suivant : http://www.qgis.org/fr/site/ 

Le plugin a été testé avec succès sur les versions 2.18.22 de QGIS.
